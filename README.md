# TB Assist

TB Assist is a desktop telemetry companion for Euro Truck Simulator 2 (ETS2) and American Truck Simulator (ATS), built with Electron, React, TypeScript, and Vite.

It provides an in-game style assistant UI for profile tools, admin/license management, telemetry-based panels, and utility workflows.

## Tech Stack

- Electron
- React + TypeScript
- Vite (via `electron-vite`)
- Tailwind CSS
- Electron Builder

## Features

- Real-time telemetry-driven dashboard experience
- Admin panel with license metadata management
- Profile and game-related tools
- Light/Dark theme support
- Windows packaging with installer output
